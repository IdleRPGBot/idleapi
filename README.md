# idleapi

API for IdleDB.


## Usage

Change each value in `.env` to your needs.

```
.env keys:

*PORT=port
*IDLEAPI_ENV=dev|prod

*PG_USER=postgres
PG_PASS=postgres
*PG_DB=postgres
*PG_HOST=localhost
*PG_PORT=5432

*REDIS_HOST=localhost
REDIS_PASS
*REDIS_PORT=6379

* = required
```
